# Scripts

Operational scripts for local development and testing.

| Script | Command | Description |
|--------|---------|-------------|
| `seed-data.js` | `npm run seed` | Seed products, inventory, and warehouse data |
| `reset-db.js` | `npm run reset-db` | Drop and recreate all collections |
| `generate-events.js` | `npm run generate-events` | Generate synthetic events |
